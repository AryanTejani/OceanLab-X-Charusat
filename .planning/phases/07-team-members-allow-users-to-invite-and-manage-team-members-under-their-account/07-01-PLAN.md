Create `backend/src/routes/team.ts` with three endpoints:

**GET /api/team** (per D-07, D-10):
- `requireAuth()` middleware, extract `userId` via `getAuth(req)`
- Query TeamMember repository: `WHERE ownerId = :userId` ordered by `joinedAt DESC`
- For each member, resolve Clerk user data:
  - If `status === 'active'` and `memberId` is not null: call `clerkClient.users.getUser(memberId)` to get `firstName`, `lastName`, `imageUrl`, `emailAddresses[0].emailAddress`
  - If `status === 'pending'`: use the stored `email`, name = "Pending", imageUrl = null
- Return `{ success: true, data: members }` where each member includes: `id, email, name (firstName + ' ' + lastName), imageUrl, role, status, joinedAt`
- Wrap each `clerkClient.users.getUser` in try/catch — if Clerk call fails, use email as name fallback

**POST /api/team/invite** (per D-01, D-02, D-08):
- `requireAuth()` middleware, extract `userId`
- Body: `{ email: string }`
- Validate email is non-empty and contains '@'
- Check for existing TeamMember with same `ownerId` + `email` — if exists, return 409 `{ error: 'Member already invited' }`
- Check that `email` is not the owner's own email — fetch owner via `clerkClient.users.getUser(userId)`, compare email, return 400 if self-invite
- Call `clerkClient.invitations.createInvitation({ emailAddress: email, redirectUrl: process.env.FRONTEND_URL || 'http://localhost:3000' })`
- Create TeamMember row: `{ ownerId: userId, memberId: null, email, role: 'member', status: 'pending', clerkInvitationId: invitation.id }`
- Return 201 `{ success: true, data: { id, email, status: 'pending', role: 'member', joinedAt } }`
- Wrap Clerk invitation call in try/catch — if Clerk returns error (e.g., already registered), still create the TeamMember row but with `status: 'active'` and attempt to find the existing user by email via `clerkClient.users.getUserList({ emailAddress: [email] })` to populate `memberId`

**DELETE /api/team/:memberId** (per D-09):
- `requireAuth()` middleware, extract `userId`
- `:memberId` is the TeamMember row `id` (UUID), NOT the Clerk userId
- Find TeamMember WHERE `id = :memberId AND ownerId = :userId` — if not found, return 404
- If `status === 'pending'` and `clerkInvitationId` is not null: call `clerkClient.invitations.revokeInvitation(clerkInvitationId)` wrapped in try/catch (ignore if already revoked)
- Delete the TeamMember row
- Return 200 `{ success: true }`
